Overview

This lab demonstrates the deployment of a cloud-native Security Operations Center (SOC) using Microsoft Sentinel in Microsoft Azure.

The objective was to simulate a monitored enterprise environment, ingest security telemetry, create detection rules, and investigate security incidents using real attack data.
